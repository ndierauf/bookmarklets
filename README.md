# bookmarklets
Handy bookmarklets.

### JiraStoryBookmarklet.js 
This is a convenience script used for copying a string containing the link to the Jira ticket and the title of the ticket. This can then be pasted into a document or email address. The result looks something like this:

"[EDA-138](https://chorusview.atlassian.net/browse/EDA-138) [EAV Modify 'setAttribute()' endpoint to include parameter for UpdateReason]"
